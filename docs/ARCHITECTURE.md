```typescript

<button onClick={() => handleStartQuest(questId)} />

  try {
    setLoading(true);
    // 3. Service makes API call
    const result = await questService.startQuest(questId);
    // 4. Update store with result
    useGameStore.setState({ currentQuest: result });
  } finally {
    setLoading(false);
  }
```

### Global State (Zustand Store)

```typescript
// Authentication State
- user: User | null
- token: string | null
- isAuthenticated: boolean
- isLoading: boolean

// Game State
- level: number
- totalXp: number
- currentQuest: Quest | null
- quests: Map<string, Quest>
- skills: Skill[]
- badges: Badge[]

// UI State
- theme: 'light' | 'dark'
- sidebarOpen: boolean
- notifications: Toast[]
```
